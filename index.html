Mini Project: African Raw Food Products Website

This project will guide you through creating a simple website for an African Raw Food Store using HTML and CSS.

Learning Objectives:

  • Structure a basic website using HTML elements.
  • Apply CSS styles to enhance the website's design and layout.
  • Showcase your creativity and personalize the website.

Project Outline:

1. Planning and Design:

  • Complete homework 6 to 9 to get the foundation for the website
  • Watch to finish styling the home, product, contact Us and login page
  • Use your own inspiration to add an about us page to your website as well as styling your signup page

2. Coding the Website:

  • Set up your environment: Use the Visual Studio Code text editor.
  • Create the HTML structure: Use all the HTML tags we have learnt and then more to define the website's basic structure like header, navigation, content sections, divs and footer.
  • Using one and only one external CSS file, style all the web pages of the website with CSS: Apply CSS styles to control the layout, fonts, colors, images, and overall look of your website.
